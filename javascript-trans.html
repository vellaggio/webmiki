Introduction and dependencies

This code can be used to create an elegant changing background with photos or color. The only thing that you have to do is decide how many photos you'll show, if the transition is gona be played repeatedly with a loop, ore just once leaving the last image/color as background, and finally you should modify your images url or define in your style the colors. You'll need jQuery


Options

Let's first assign those variables with options you like: 

		
<script>
	var i=1;
	var num_img= ;/*assign here number of images or colors*/
	var loop=true; /*set to false if you don't want to repeatedly play the transition*/
</script>

Javascript functions

		
<script>
	function ascendOP(){
		var img = "#img_bg"+i.toString()+"";	
		$(img).animate({			
			opacity: 0.9			
		}, 7000, function() {
			descendOP();			
		});
	}
	
	function descendOP(){
		var img = "#img_bg"+i.toString()+"";					
		if(i!=num_img)
		{
			i++;
			ascendOP();					
		}else if(loop)
		{
			i=1;
			ascendOP();
		}
		$(img).animate({			
			opacity: 0			
		}, 7000, function() {
		});
	}
</script>

Start the transition by calling the function descendOp() 

		
<script>
	$('document').ready(function(){
		descendOP();
	});
</script>

Using it with images

You have to put your html code in the place that you want. If you will use it as background I recommend to put it just before the body tag, inside a div with full width and height, for example with 4 images: 

		
		
<div id="bg">	
	<div id="img_bg4"><img src="path/to/your/image/image.jpg"></div>
	<div id="img_bg3"><img src="path/to/your/image/image.jpg"></div>
	<div id="img_bg2"><img src="path/to/your/image/image.jpg"></div>
	<div id="img_bg1" style="opacity:1;"><img src="path/to/your/image/image.jpg"></div>
</div>			

		
					
#bg {
  position: fixed; 
  top: 0; 
  left: 0; 
  min-width: 100%; 
  min-height: 100%;
  z-index:-1;
}

#bg div img{
  width: 100%; 
  height: 100%;
}

Using it with colors

To display colors we can use css classes 

		
		
<div id="bg">		
	<div id="img_bg4" class="bg_color bg_color-red"></div>
	<div id="img_bg3" class="bg_color bg_color-green"></div>
	<div id="img_bg2" class="bg_color bg_color-blue"></div>
	<div id="img_bg1" class="bg_color bg_color-black" style="opacity:1;"></div>	
</div>		

		
					
#bg {
  position: fixed; 
  top: 0; 
  left: 0; 
  min-width: 100%; 
  min-height: 100%;
  z-index:-1;
}

.bg_color {
  position: absolute; 
  top: 0; 
  left: 0; 
  right: 0; 
  bottom: 0; 
  margin: auto; 
  min-width: 100%;
  min-height: 100%;
  opacity:0;
}

.bg_color-blue{
	background-color: rgba(19,183,211, 1);
}

.bg_color-black{
	background-color: rgba(0, 0, 0, 0.95);
}

.bg_color-red{
	background-color: red;
}
.bg_color-green{
	background-color: green;
}
